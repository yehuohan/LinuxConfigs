# LinuxConfigs
 管理Linux的一些配置文件，包括vim, zsh, Msys2等配置文件


---
# Files

```
.
├── .git
├── cf-msys2               : Win-Msys-config
│   └── .minttyrc          : Msys2模拟终端mintty配置文件
├── cf-vim                 : Vim-config
│   ├── .vimrc             : vim配置文件
│   ├── .ycm_extra_conf.py : vim-ycm配置文件
│   ├── autoload           : vim-plug插件
│   └── mySnippets         : 自定义snippets
├── cf-zsh                 : zsh-config
│   ├── .zshrc             : zsh配置文件
│   └── frisk.zsh-theme    : zsh主题文件
```
